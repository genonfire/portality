  • - 뉴스 등록하기
  • - 투표하기와 베스트
  • - 회원가입 및 로그인
  • - 기자 DB
  • - 버스터 콜

기자 DB


DB에 등록된 기자들을 찾을 수 있고(2) 한꺼번에 볼 수도 있습니다.(3)

매달 생성되는 명에의 전당 링크(1)에서는 해당 월에 가장 많이 나빠요!를 받은 기자들의 순위를 보여줍니다.



기자의 이름을 선택하면 기자 정보를 편집(1) 할 수 있는데 로그인 상태에서만 가능합니다.
작성 기사와 언론사 기사 링크(2)는 해당 기자나 해당 기자의 소속 매체에서 작성한 기사를 보여줍니다.
언론사 이름(3)을 누르면 해당 매체에 소속된 모든 기자들을 찾아 보여 줍니다.



기자 추가


기자 추가는 로그인 상태에서만 가능합니다. DB에 존재하지 않는 기자는 기자 추가 버튼을 눌러 추가할 수 있습니다.



기자를 추가하거나 편집하면 수정한 사람의 아이디가 기록됩니다. 허위 사실이나 명예 훼손이 될만한 것이나 모욕을 줄 수 있는 내용은 작성하지 않도록 주의해 주세요.



편의 기능


기자 정보를 조금 더 편하게 추가하는 방법이 있습니다.

노룩뉴스 페이지에서 기자의 이메일 주소를 누릅니다.



해당 기자 정보가 DB에 없으면 검색한 이메일 주소로 생성이 가능합니다.



이메일 주소와 소속이 자동으로 입력됩니다. 입력된 정보를 확인하고 나머지 정보를 입력 후 저장하면 됩니다.
