뉴스 등록하기


1. 아래 링크에서 크롬 확장 프로그램 노룩뉴스 - 버스터 콜을 설치합니다.
https://goo.gl/Mxdjy9



2. 인터넷 뉴스를 보다가 나빠요! 혹은 좋아요!를 표시하려면 기사 하단에 있는 기자의 이메일 주소를 복사해 둔 후(1)



페이지 상단의 기사 제목을 드래그해 선택하고(2) 확장프로그램 아이콘을 누르면(3) 기사 제목과 주소가 자동으로 입력됩니다.(4)
기자 이메일 칸에 아까 복사해 둔 이메일 주소를 붙여넣기 합니다.(5) 이메일 주소는 없으면 안 적어도 됩니다.
좋아요!로 표시하려면 좋아요!에 체크하세요.(6)



찍기 버튼을 누르면(7) 서버로 정보를 보낸 후 성공 메세지가 나옵니다.(8)
노룩뉴스 링크를 누르면 http://nolooknews.com 사이트로 이동합니다.(9)




사이트에서 직접 등록하기


1. 크롬 확장 프로그램을 사용할 수 없는 환경에서는 웹사이트에서 직접 등록할 수 있습니다.
2. 로그인한 상태에서만 가능합니다. 회원가입 방법 보기
3. 나빠요! 를 추가할지 좋아요!를 추가할지 선택합니다.(1)
4. 이슈 생성 버튼을 누르면 선택한 뉴스를 생성합니다. (2)



5. 기사 제목, 주소, 이메일을 입력합니다. (3)
6. 나중에 기사가 삭제돼도 볼 수 있도록 박제를 해주는 http://archive.is의 스냅샷 주소가 있다면 적어줍니다.(4)
7. 제출 버튼을 누르면 이슈가 생성됩니다.(5)



8. 이미 존재하는 뉴스를 생성할 경우 새로 생성되지 않고 나빠요! 혹은 좋아요! 숫자가 +1 됩니다.
